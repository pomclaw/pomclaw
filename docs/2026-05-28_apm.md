调用链路

1. 启动时（servicecontext.go）

OTelPGExporter (写 traces/spans 表)                                                              
↓
LocalTracerProvider (WithSyncer = span.End() 时同步 export)
↓
opentelemetry.SetProvider(traceProvider, meterProvider)
↓ 存入 pkg/eino_opentelemetry 的 providerRegistry

apmplus.NewApmplusHandler(...)
→ 内部调 opentelemetry.NewOpenTelemetryProvider(...)
→ 命中 providerRegistry，返回上面注册的 traceProvider
→ apmplusHandler.tracer = traceProvider.Tracer(scope)  ✅ 已绑定 OTelPGExporter
↓
callbacks.AppendGlobalHandlers(apmplusHandler)  ← 注册为全局 callback

2. 每次请求（loop.go）

ctx = apmplus.SetSession(ctx, sessionID, userID)   ← session 注入 context
↓
runner.Run(ctx, messages, adk.WithCallbacks(streamCallback))
↓
[Eino 对每个组件自动触发全局 + 局部 callback]
↓
apmplusHandler.OnStartWithStreamInput(ctx)
→ a.tracer.Start(ctx, spanName) 创建 span，存入 ctx
↓
[LLM/Tool 运行]
↓
apmplusHandler.OnEndWithStreamOutput(ctx)
→ goroutine: 消费流 → span.End()
↓
SimpleSpanProcessor.OnEnd(span)  ← WithSyncer 同步触发
↓
OTelPGExporter.ExportSpans → exportSpan
→ ensureTrace → INSERT traces (首次) / FindByOtelTraceID
→ INSERT spans
→ UPDATE traces 聚合字段
